# vue-cli-plugin-esbuild

> Vue CLI plugin to use [esbuild](https://esbuild.github.io/) for js/ts compilation and minification

## 🚀 Install

```sh
vue add esbuild
```
or
```sh
npm install --save-dev vue-cli-plugin-esbuild
```
